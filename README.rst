micmac
======

Old-school, SDL-driven application built around MacCormack's method for nozzle design CFD

Requires the following co-located DLLs:
 * jpeg.dll (libjpeg)
 * libfreetype-6.dll (FreeType)
 * libogg-0.dll (OGG)
 * libpng12-0.dll (libPNG)
 * libtiff-3.dll (TIFF)
 * libvorbis-0.dll (Vorbis)
 * libvorbisfile-3.dll (Vorbis File)
 * mikmod.dll (MikMod sound library)
 * SDL.dll (SDL v1?)
 * SDL_image.dll (SDL Image)
 * SDL_mixer.dll (SDL Mixer)
 * SDL_ttf.dll (SDL TTF)
 * smpeg.dll (SMPEG)
 * zlib1.dll (Z-Lib)
